### **1.Decode 函数**
#### **說明：**
```SQL
Decode(表達式, 條件1, 結果1, 條件2, 結果, 其他)
```
#### 實例：
表名: product
| ID     | item     | price    | 
| :----: | :----:   | :----:   | 
| A01    | 口香糖    | 10       |
| A01    | 茶葉蛋    | 8        |
| A02    | 香菸      | 80       |
| A02    | 報紙      | 10       |
```SQL
Select ID, item, Decode(price, '10', '十', '80', '八十', '其他') From product
```
#### 輸出：
| ID     | item     | price    | 
| :----: | :----:   | :----:   | 
| A01    | 口香糖    | 十       |
| A01    | 茶葉蛋    | 其他     |
| A02    | 香菸      | 八十     |
| A02    | 報紙      | 十       |
